# Screenshots Directory

This directory contains screenshots and GIFs of the PokéDex application.

## Directory Structure

```
screenshots/
├── gif/
│   ├── demo.gif              # Main app demo GIF
│   ├── search.gif            # Search functionality demo
│   └── navigation.gif        # Navigation flow demo
│
├── home_screen.png           # Home screen screenshot
├── horizontal_scroll.png     # Horizontal scrolling view
├── empty_search.png          # Empty search state
├── search_bar.png            # Search bar view
├── search_results.png        # Search results view
├── clear_search.png          # Clear search action
├── detail_screen.png         # Pokemon detail screen
├── stats_view.png            # Stats section view
├── abilities_view.png        # Abilities section view
├── loading.png               # Loading state
├── error.png                 # Error state
└── pull_refresh.png          # Pull to refresh action
```

## Adding Screenshots

1. Take screenshots from your device/emulator
2. Save them with the appropriate names listed above
3. Place them in this directory
4. Update the README.md if you use different names

## Adding GIFs

1. Create GIFs using screen recording tools (e.g., ScreenToGif, LICEcap)
2. Save them in the `gif/` subdirectory
3. Keep file sizes reasonable (< 10MB recommended)
